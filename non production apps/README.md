# Apps

This folder contains complete applications for Business Central, provided by Microsoft but maintained by the community.

## Important Notice

⚠️ **These applications are provided "as-is" without official support.**

- **No warranties**: These apps are provided without any guarantees or warranties of any kind
- **Community maintained**: While initially provided by Microsoft, these apps are maintained by the community
- **Use at your own risk**: Test thoroughly in your own environment before using in production
- **No official support**: Microsoft Support will not provide assistance with these applications

## Purpose

Unlike the `/samples` folder which contains code samples and snippets for learning purposes, the `/apps` folder contains full, working applications that you can deploy and use. These apps can serve as starting points for your own projects.
